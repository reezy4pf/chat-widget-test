Troubleshooting Guide

If you're experiencing issues with the chat widget:

  • 500 Server Error: The webhook URL is returning a server error (HTTP 500). Please verify that the n8n instance at yoreezy.app.n8n.cloud is running properly and that the webhook is correctly configured.
  • Webhook Configuration: Verify that the webhook expects the payload format shown in the console logs.
  • CORS Issues: Make sure the n8n webhook has CORS enabled for this domain.
  • N8N Workflow: Check that the webhook workflow in n8n is properly configured to respond to the "loadPreviousSession" action.
